# Data-Warehouse
Data Portfolio Creando un Data Warehouse con SQL Server, ETL , Análisis y Data modeling.
Proyecto base para la creacion de un Data Warehouse. Usaré este proyecto para demostrar la compresión en como manipular y analizar la data. Una de las mejores formas de estudiar Data anaylsis y Data Ingenieria usando SQL Server y Visual Studio para debugging y mejorar su lectura.

--- 


## Proyecto --

### Objetivo --
El objetivo es desarrollar con éxito con mis habilidades con usando SQL Server para el codigo una Data Warehouse usando el metodo de 3 procesos [BRONZE, SILVER, GOLD] y profundizar con Datasets para Análisis y la toma de desicíon para ''tal empresa''. Una manera de mejorar el razonamiento para situaciones o anomalias encontradas en el proceso de Data Cleaning.

#### Especificaciones

## Data Fuente ##: Importar data de dos fuentes (ERP y CRM) proveniente de CSV archivos.
## La calidad de la Data ##: Limpieza y resolver su calidad antes de analizarla.
## Integracion ##: Combina las dos fuentes en una, una fuente de data desiñada para analitica de las Queries.
## Visíon ##: Foco on la ultima fase (dataset); crear un logging en esta fase no será necesario.
## Documentacion##: Proviene documentacion limpia del modelo de Data para brindar soporte al equipo que sepa manipular y consultar Data
